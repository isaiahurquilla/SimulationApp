# SimulationApp

This project is a React Native app made with Expo for CMPS 3390 Lab 05. It has 4 main parts in one app:

- Dashboard
- Stock Ticker
- Name Generator
- Oil Pump Monitor

The app is meant to practice using `useState`, `useEffect`, conditional rendering, and rendering lists with `.map()`.

## What you need

Before running the project, install:

- Node.js
- Git
- VS Code
- Expo Go on your phone or Android Studio for emulator

## How to set it up

Clone the repo and install dependencies:

git clone <your-repo-link>
cd SimulationApp
npm install

## For running it on your Browser
Install: npx expo install react-dom react-native-web
Terminal: npx expo start
